== Description ==

**Newsletters, Email Marketing, Automation and CRM for WordPress**

SureContact is a cloud-based email marketing and CRM platform built specifically for WordPress.

It lets you send newsletters, run email automations, manage contacts and track ecommerce revenue from one dashboard, without slowing down your site.

Instead of relying on expensive external tools or overloading your WordPress hosting, SureContact keeps execution in the cloud while staying deeply connected to WordPress.

With SureContact, you can:

* Manage all contacts and customers in one place
* Automatically sync data from WooCommerce, SureCart and forms
* Send newsletters and automated emails reliably via your own SMTP
* Track revenue generated from campaigns and automations
* Run everything without adding load to your WordPress site

You get the simplicity of WordPress with the power and reliability of a modern email marketing platform.

That's what **SureContact** is built for.

**External Service Connection / Security**

SureContact connects your WordPress site with our SaaS securely. When a user initiates a connection, they are redirected to the SaaS login page. Upon clicking Connect, the SaaS generates a short-lived, single-use token which is sent back to WordPress via a redirect. WordPress then verifies the token server-to-server before storing the access token and user data for future API calls. Tokens are time-limited (1–5 minutes), single-use, and encrypted in transit using HTTPS. This ensures secure authentication, prevents token reuse, and protects user data. No actions are taken by WordPress until token verification succeeds. The access token will be used to sync contact data from various sources including WordPress users, e-commerce platforms (SureCart, WooCommerce, Easy Digital Downloads), and form submissions (SureForms, WPForms, Contact Form 7) to your SureContact workspace. All privacy and data handling comply with WordPress.org Plugin Guidelines.

== Installation ==

**Automatic Installation (Recommended)**
1. Log in to your WordPress admin dashboard
2. Navigate to Plugins > Add New
3. Search for "SureContact"
4. Click "Install Now" and then "Activate"
5. Click "Get Started Now" to connect your SureContact workspace
6. Complete the authentication process
7. Enable the integrations you want to use (e-commerce, forms, etc.)

**Manual Installation**
1. Download the SureContact plugin
2. Upload the plugin files to `/wp-content/plugins/surecontact/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Navigate to SureContact in your admin menu
5. Click "Get Started Now" to connect your WordPress site to SureContact
6. Complete the authentication process to link your workspace
7. Configure your integration settings

**After Installation**
Once connected, SureContact will automatically start capturing contact data from your enabled sources. You can manage your contacts, view interaction history, and analyze customer behavior from your [SureContact Dashboard](https://app.surecontact.com/)

== Frequently Asked Questions ==

= Can I send unlimited emails with SureContact? =
Yes. SureContact doesn't charge based on the number of emails you send. You can send unlimited newsletters and automated emails.

= Is SureContact free? =
Yes. SureContact offers a free-forever plan with all core features, up to 250 contacts and 500/month email sends.

= Can SureContact slow down my WordPress site? =
No. Because campaigns, automations and syncing run in the cloud, your site performance remains unaffected.

= Do I need an SMTP provider to send emails? =
Yes. SureContact connects to SMTP providers such as SendGrid, AWS SES, Mailgun, Gmail, or any standard SMTP service. This gives you full control over email delivery.

= Can I manage multiple sites or brands with SureContact? =
Yes. SureContact allows you to manage multiple brands or client accounts from a single workspace, making it suitable for agencies and consultants. However, we recommend setting up dedicated workspaces for each site.

= Where can I get help if I need support? =
You can access the SureContact knowledge base, post in the WordPress plugin support forum, or contact the support team directly.

= How can I report a security bug? =
We take plugin security extremely seriously. If you discover a security vulnerability, please report it in a safe and responsible manner.

You can report the issue through our [Bug Bounty Program](https://brainstormforce.com/bug-bounty-program/). We collaborate with Patchstack to provide opportunities for researchers to report vulnerabilities. The Patchstack team will help validate, triage, and handle any reported security issues.
